# lyl_sc
项目介绍：
  一个基于多进程的分布式爬虫，爬取新浪，天涯社区，人民网等
项目依赖包:
- python version==2.7
- pip install django==1.9.2
- pip install MsSQL-python 
- pip install redis
- pip install paramiko
- pip install multiprocessing 
- apt-get install lxml
- pip install dwebsocket
- pip install selenium selenium
